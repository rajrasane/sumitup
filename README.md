# SumItUp - AI Powered PDF Summarizer

SumItUp is a modern web application that uses advanced AI technology to transform lengthy PDF documents into clear, concise summaries, saving you hours of reading time.

## Features

- **PDF Upload**: Easily upload your PDF documents
- **AI-Powered Analysis**: Our intelligent system processes and extracts key information
- **Instant Summaries**: Receive clear, actionable summaries in seconds
- **User Authentication**: Secure account management via Clerk
- **Responsive Design**: Beautiful interface that works on all devices

## Technology Stack

- Next.js 15.3.0 with React 19
- TypeScript for type safety
- Tailwind CSS for styling
- Clerk for authentication
- Vercel for analytics and deployment

## Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/rajrasane/sumitup
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run the development server
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application
